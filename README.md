# Personal Portfolio — David Oloan Sitorus

A clean, responsive personal portfolio site built with vanilla HTML, CSS, and JavaScript.

## Features

- Responsive single-page layout with smooth scroll animations
- Graduation photo slideshow in the hero section
- Sections: About, Skills, Projects, Contact

## Tech Stack

HTML · CSS · JavaScript

## Getting Started

No build tools or dependencies needed — just open `index.html` in your browser.

git clone https://github.com/cocobubur/personal-portfolio-site.git
cd personal-portfolio-site
open index.html
